How to run the Customer Complaint System 

1. Download the  zip file

2. Extract the file and copy Customer_Complaint_System folder

3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

4. Open PHPMyAdmin (http://localhost/phpmyadmin)

5. Create a database with name app

6. Import app.sql file(given inside the zip package in SQL file folder)

7.Run the script http://localhost/Customer_Complaint_System (frontend)

8. For admin panel

http://localhost/Customer_Complaint_System/Admin  (admin panel)
